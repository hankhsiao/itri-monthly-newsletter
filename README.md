# ITRI Monthly Newsletter

A curated tech information newsletter for ITRI (Industrial Technology Research Institute) members in Silicon Valley, delivered monthly with clean, professional design and comprehensive coverage of industry trends.

## Project Goal

Create a modern, responsive web application to deliver a monthly newsletter featuring curated technological information and industrial events to ITRI members. The newsletter provides centralized access to relevant tech insights across key strategic areas.

## Requirements

### 1. Overall Requirements
- **Responsive Design**: Fully functional on mobile and desktop devices
- **Email Version**: Support for separate email-compatible format for distribution
- **Theme Support**: Light and dark mode compatibility (web version only)
- **Design Philosophy**: Information-centric, clean, professional interface for easy information access

### 2. Content Structure
- **Single Page Layout** containing:
  - Content table (with navigation to sections and subsections)
  - Tech information organized by 4 main categories (with subcategories)
  - Industrial events section
  - Footer with ITRI contact information

- **Tech Information**:
  - Article summaries with: title, optional thumbnail, source, date, and link to original
  - Organized by 4 main categories with relevant subcategories

- **Industrial Events**:
  - Conference events with: title, date, location, event link, and directions link

- **Footer**:
  - General contact information for ITRI

### 3. Tech Categories

**A. 智慧生活 (Smart Living)**
- A1. 個人化裝置與服務 (Personalized Devices & Services)
- A2. 自主移動系統 (Autonomous Mobility Systems)
- A3. 智慧產業及服務 (Smart Industry & Services)

**B. 健康樂活 (Health & Wellness)**
- B1. 智慧醫療 (Smart Healthcare)
- B2. 健康照護 (Health Management)

**C. 永續環境 (Sustainable Environment)**
- C1. 循環經濟 (Circular Economy)
- C2. 智慧製造 (Smart Manufacturing)
- C3. 綠能系統與環境科技 (Green Energy & Environmental Tech)

**T. 智慧化致能技術 (Intelligent Enabling Technology)**
- T1. 人工智慧技術 (AI Technology)
- T2. 半導體晶片技術 (Semiconductor Chip Technology)
- T3. 通訊技術 (Communication Technology)
- T4. 資安與雲端技術 (Cybersecurity & Cloud Technology)

## Getting Started

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the newsletter.

The app will automatically reload as you make changes to files in the `app/` directory.
